数据图表样式库

35种核心图表类型 — hover交互 + 入场动画

折线图 Line Chart

展示数据随时间的变化趋势

柱状图 Bar Chart

比较不同类别的数值大小

饼图 Pie Chart

展示各部分占总体的比例

面积图 Area Chart

强调数据变化的累积量

散点图 Scatter Plot

展示两个变量之间的相关性

雷达图 Radar Chart

多维度能力对比

横向柱状图 Horizontal Bar

水平排列条形，适合类别名较长的数据对比

堆叠柱状图 Stacked Bar

每个柱由多段堆叠组成，展示分量与总量的关系

分组柱状图 Grouped Bar

每组内多个柱并列，便于组内细项对比

象形图 Pictogram

用重复图标数量直观表示数值大小

面积堆叠图 Stacked Area

多层面积叠加，展示总量变化和各部分贡献

迷你图 Sparkline

紧凑趋势线，适合在文字或表格内嵌入展示

阶梯图 Step Chart

阶梯状连线，适合展示离散变化和利率调整

环形图 Donut Chart

饼图中心挖空放置总计，视觉更轻盈

漏斗图 Funnel Chart

上宽下窄，展示各阶段转化与流失

旭日图 Sunburst Chart

多层同心环形，展示层级数据的占比关系

树状图 Treemap

矩形嵌套表示层级占比，空间利用率高

玫瑰图 Nightingale Rose

扇形半径表示数值，极坐标下的面积对比

气泡图 Bubble Chart

散点加气泡大小表示第三维度数据

维恩图 Venn Diagram

圆形重叠区域表示集合间的交集关系

象限图 Quadrant Chart

十字分四区加散点，用于战略定位分析

平行坐标图 Parallel Coordinates

多条平行纵轴连线，展示多维数据模式

箱线图 Box Plot

显示最小值、Q1、中位数、Q3、最大值五数概括

小提琴图 Violin Plot

对称核密度加内嵌箱线，展示分布形态

直方图 Histogram

相邻柱无间距，展示数据的频率分布

密度图 Density Plot

平滑曲线表示数据分布的概率密度

蜂群图 Beeswarm

散点不重叠排列，展示每个数据点的分布

桑基图 Sankey Diagram

宽度变化的流向带，展示流量或能量的转移

瀑布图 Waterfall Chart

逐级增减柱，展示从起点到终点的过程

弦图 Chord Diagram

圆形外侧节点加弧线连接，展示流向关系

仪表盘 Gauge

半圆弧刻度加指针，展示单一指标的位置

进度条 Progress Bar

横向填充条加百分比，展示完成进度

热力图 Heatmap

色块矩阵表示交叉维度的数值大小

K线图 Candlestick

蜡烛形状展示开盘、收盘、最高、最低价

极坐标柱状图 Polar Bar

从圆心向外辐射的柱，展示周期性数据
