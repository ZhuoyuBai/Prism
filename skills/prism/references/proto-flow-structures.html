35种核心流程结构 — 点击节点查看详情

Linear 线性流程

单向串联，适合步骤明确的顺序流程

Parallel 并行流程

多路同时执行后汇聚，适合独立子任务并行处理

Branch 条件分支

根据条件走不同路径，适合判断逻辑

Loop 循环流程

重复执行直到满足条件，适合迭代优化场景

Nested 嵌套层级

外层流程包含内层子流程，适合复杂系统的分层展示

Interactive 交互式流程

支持点击节点查看详细信息，适合教学讲解
